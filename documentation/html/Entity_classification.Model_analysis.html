Module contents

This module has implementation of various model analysis and their evaluation. : Module with implementation of various OOD detection methods and their evaluation.

Dirs:

** Data_distribution ** : module for doing the comparison of similarity between data distribution of different models

** Data_representation ** : module for doing the comparison of cosine similarity between OOD detection

** Error_analysis ** : module to compare the error profiles

** resources ** : Directory containing the embeddings, predictions, softmax scores and groundtruth of the train, eval and test datasets that are copied manually from “generate_embeddings” folder.
