Module contents

This module is used generate embeddings for the threes tasks using the four models - vanilla , DisEnt, Feature_sieve, LfF. Embeddings are automatically saved inside ‘[Task]/[ModelName]/Embeddings’ folder, which can be copied wherever required.

dirs:

Entity_classification : codes to generate , train and test four models for Entity Classification Task

NLI : codes to generate , train and test four models for NLI Task

Paraphrase_identification : codes to generate , train and test four models for Paraphrase Identification Task.
